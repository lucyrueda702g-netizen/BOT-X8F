# 🤖 MXYKEL Discord Bot

Bot de Discord con bienvenidas, despedidas y 50+ comandos.

## 🚀 Subir a Railway

1. Ve a **railway.app** y crea cuenta
2. Toca **"New Project"** → **"Deploy from GitHub"**
3. Sube estos archivos a GitHub
4. En Railway ve a **Variables** y agrega:
   - `TOKEN` = tu token del bot
5. Railway lo inicia automáticamente

## ⚙️ Comandos disponibles

### Auto Config
- `/setup` — Crea todos los canales automáticamente

### Bienvenidas/Despedidas
- `/setwelcome` — Canal de bienvenidas
- `/setfarewell` — Canal de despedidas
- `/setlogs` — Canal de logs
- `/testwelcome` — Probar bienvenida
- `/testfarewell` — Probar despedida

### Info
- `/userinfo` `/serverinfo` `/avatar` `/ping` `/botinfo`
- `/roleinfo` `/channelinfo` `/membercount` `/stats`

### Moderación
- `/ban` `/kick` `/mute` `/unmute` `/warn` `/unban`
- `/clear` `/lock` `/unlock` `/slowmode`
